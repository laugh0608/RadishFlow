# RadishFlow v26.5.1-dev Release Notes

更新时间：2026-05-16

## 用途

用途：记录 MVP α `v26.5.1-dev` 候选 tag 的发布说明、验证结果和便携包边界。
读者：准备执行 MVP α 内部验收、创建 tag 或归档便携包的开发者、验证人员和 AI / Agent。

## 发布定位

`v26.5.1-dev` 是 MVP α 内部集成验收候选版本。当前交付形态是 Windows 便携压缩包 / staging 目录，不是安装器。

候选包入口：

```text
artifacts/packages/RadishFlow-v26.5.1-dev-windows-x64/
artifacts/packages/RadishFlow-v26.5.1-dev-windows-x64.zip
```

包内启动入口：

```powershell
.\radishflow-studio.exe
```

## 主要内容

- Rust Studio MVP α 最小工作台：打开示例、新建空白、最短建模、运行、审阅结果、保存 / 另存为和重开项目。
- Canvas 覆盖 `Feed -> Flash Drum`、`Feed -> Heater/Cooler/Valve -> Flash Drum`、`Feed + Feed -> Mixer -> Flash Drum` 三条最短可求解路径。
- Result Inspector / Active Inspector 只读消费同一份 `SolveSnapshot`，可审阅 stream-centric / unit-centric 结果、相结果、`bubble_dew_window`、overall enthalpy、关联步骤和诊断目标。
- Stream Inspector 支持 composition draft、显式提交、normalize、discard、受控组件添加 / 删除和运行前阻断。
- 中文界面资源覆盖 MVP α smoke 高频路径。
- Windows 便携包包含 Studio exe、正向 flowsheet 示例、样例物性包、quick start、结果审阅说明、验收清单、版本说明、release notes 和许可文件。
