# Flowsheets

该目录存放示例流程。

- `feed-mixer-flash.rfproj.json`: 当前最小可求解示例，覆盖 `Feed1 + Feed2 -> Mixer -> Flash Drum`
- `feed-heater-flash.rfproj.json`: 第二条最小可求解示例，覆盖 `Feed -> Heater -> Flash Drum`
- `feed-cooler-flash.rfproj.json`: 第三条最小可求解示例，覆盖 `Feed -> Cooler -> Flash Drum`
- `feed-valve-flash.rfproj.json`: 第四条最小可求解示例，覆盖 `Feed -> Valve -> Flash Drum`
